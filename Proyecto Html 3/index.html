Fundamentos de Programacion

Información general

Un programador de aplicaciones crea software que será ejecutado por un dispositivo programable: computadora, teléfonos, reloj, entre otros. Antes de crear una aplicación, ya sea el FrontEnd o el BackEnd, un programador debe tener muy claro los conceptos más fundamentales de la programación, que seran aplicados en cualquier lenguaje.

Información general

Es por esto, que el profesional del soporte técnico es indispensable en cualquier organización que posea 2 computadoras o más, para ofrecer apoyo en las operaciones diarias de la organización, al dar una respuesta rápida a cualquier falla o necesidad en los equipos de tecnologias de información.

Programación BackEnd con PHP

Información general

Un BackEnd es la parte de una aplicación que le da respuestas a los distintos FrontEnds de una plataforma. El BackEnd no es visible para el usuario final y se puede programar en cualquier lenguaje del lado del servidor. Php es un lenguaje muy utilizado para crear páginas web dinámicas. Como se ejecuta del lado del servidor, también es muy utilizado para crear un BackEnd que de soporte a los FrontEnd que lo requieran.

Objetivo general

Crear el BackEnd de una aplicación Web utilizando el FrameWork Laravel, que implemente servicios Web REST, que se conecte a una base de datos Mysql.
